Description

A significant part of the Random forest model is the bootstrap sampling. We assume you are already familiar with the concept, so we will make a simple reminder: to make this sampling, we randomly sample our dataset with replacement.

In this stage, create a function with the data as a parameter and return a bootstrap sample. We recommend using the np.random.choice to create a mask and then return only the masked objects. You can take this approach as an example. We expect that the function is called something like create_bootstrap.

In this stage, you'll deal with randomness. For the sake of our tests, use the np.random.choice function once with the 52 seed.
Objectives
  1. Create the create_bootstrap function;
  2. Use it to create a bootstrap sample from your train data and print the first ten items of the resulting sample y-s.
